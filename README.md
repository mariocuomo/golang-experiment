# Golang Experiment
The goal of this repository is to experiment with the **Golang** language and create a library of **sorting algorithms**.

Currently available (_int list_ as parameter)
- **STUPID SORT**
- **SELECTION SORT**
- **MERGE SORT**
- **BUBBLE SORT**
- **TREE SORT**
